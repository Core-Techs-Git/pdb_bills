@core-techs-git/pdb_bills

PDB BILLS

This module gather in one place all the dependencies required by La Plateform du bâtiment projects to interact with bills document. It relays on external services like Docapost.

Features

Below are the features offered by pdb_bills:

  • Search in stored documents
  • Retrieve a specified document

How to use it ?
  1. Install with npm
$ npm i -S @core-techs-git/pdb_bills

⚠️ To install this way you need to have configured npm to use github package registry. To know more you may follow this link.

Or you can follow this quick setup

  • Set a scoped registry access

    $ npm config set @core-techs-git:registry https://npm.pkg.github.com/core-techs-git

  • Set authentication information inside your .npmrc file

    //npm.pkg.github.com/:_authToken=PERSONAL-ACCESS-TOKEN

PERSONAL-ACCESS-TOKEN is generated in github settings.

  1. Edit your configuration file config.js (located at the root of your project) according to pdb_requester since bills depends on it to make its requests.
const config = {};
...
config.requester = {
  ...
  docapost: {
    proxy: false,
    protocol: 'https'
  },
  ...
};
...
module.exports = config;
  1. Defined need environment variables for Docapost service
Variable Description
DOCAPOST_HOST The docapost service host string
DOCAPOST_PATH Path to the service end point. start with /
DOCAPOST_USER The login string
DOCAPOST_PASSWORD The password string
  1. Import the module inside your code and just use it
const bills = require('@core-techs-git/pdb_bills');

docapost.serviceDoc(docId, (err, data) => {
  console.log('err', err);
  console.log('data', data);
});

docapost.serviceSearch({
  code_depot: '2',
  priceFrom: '0',
  priceTo: '10.6'
}, (err, data) => {
  console.log('err', err);
  console.log('data', data);
});

How to contribute ?
  1. Import the project from github
$ git clone git@github.com:Core-Techs-Git/pdb_bills.git
  1. Install dependencies with npm
$ npm i

That's it you're all setup and can start contributing :thumbsup:.
